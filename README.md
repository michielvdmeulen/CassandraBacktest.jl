## Installation
To install `CassandraBacktest.jl`, press `]` to activate Package mode and type the following command:

```julia
pkg> add CassandraBacktest
```

## Quick start
Assuming `CassandraBacktest.jl` has been installed, start using the package as follows:

```julia
using CassandraBacktest
```

See package documentation for more syntax options.
